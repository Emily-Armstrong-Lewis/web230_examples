WebSocket Example

To use this example you must first open the broadcast.py file using python (python broadcast.py NOTE: Use python 2.7 not python 3)

Then open another copy of this page into the browser. On both pages click the connect button to open the socket. You will see the status change to "Socket Open." Enter some text in the box and click "send." You will see a message on the sending end that shows the message sent and received, on the other page you will see that the message sent is just received but not sent. You can open an many browser tabs as you want.
